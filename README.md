# diplom_2026

Инструкция по начальной настройке

1.	Обновление исходной конфигурации.

1.	Запустить конфигуратор;
2.	Снять конфигурацию с поддержки. Перед началом любых действий с конфигуратором 1C рекомендуется выгрузить информационную базу 1C в файл. Выбрать меню «Администрирование» далее «Выгрузить информационную базу».
3.	Выбрать пункт меню «Конфигурация» далее «Сравнить, объединить с конфигурацией из файла…».
4.	Выбрать необходимый файл новой конфигурации с расширением. cf и дождаться окончания процесса сравнения.
5.	Результат сравнения и объединения будет открыт в специальном окне, в котором показана структура конфигураций.
6.	Для объединения конфигураций нажать на кнопку «Выполнить», дождаться сообщения об успешном объединении и нажать на кнопку «ОК».
7.	Далее необходимо обновить конфигурацию базы данных. После появления окна со списком измененных объектов нажать на кнопку «Принять».

2.	Создание пользователей.

1.	Запустить1С: Предприятие в режиме «Отладка».
2.	Зайти в подсистему «Администрирование», выбрать раздел «Настройки пользователей и прав» далее «Пользователи».
3.	Создать пользователей со следующими правами (Выбрать во вкладке «Права доступа»):
- Администратор (если еще не создан)
- Специалист
- Менеджер
- Бухгалтер
- Кадровик-расчетчик

3.	Заполнить информацию о сотрудниках.

-  по оплате труда в регистре сведений «Условия оплаты сотрудников», 
- график отпусков в документе «График отпусков»,
- размер премии в документе «Начисление фиксированной премии».

4.	Заполнить график работ.

- Заполнить справочник «График работы» (Пятидневка или Шестидневка)
- С помощью обработки «Заполнение графика» заполнить справочник. Выбрать график работы (Пятидневка или Шестидневка), указать необходимый период и выходные дни недели.




5.	Настройка интеграции с Telegram.

1.	Создайте бота Telegram:
- Напишете в Телеграм https://t.me/BotFather команду "/start"
- Напишите команду "/newbot"
- Выберите имя вашего бота, которое будут видеть пользователи
- Выберите идентификатор вашего бота (он должен заканчиваться на "bot")
- Скопируйте токен, который пришлет BotFather и сохраните его в константу «Токен управления телеграм-ботом»

2.	Создание группы в Telegram:
- Создайте группу в Телеграм
- Добавьте в группу только что созданного бота
- Назначьте боту права администратора

3.	Получите идентификатор группы:
- Напишите любое сообщение в группу
- С помощью браузера или Postman выполните запрос https://api.telegram.org/bot[ВашТокен]/getUpdates. В полученном JSON найдите идентификатор группы, в которой получено сообщение, и сохраните его в константу «Идентификатор группы для оповещения».

6.	Заполнить константы для документов «Реализация товаров и услуг».

 «Номенклатура абонентская плата» и «Номенклатура работы специалиста» заполняются из справочника «Номенклатура».
